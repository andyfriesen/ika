ikaMap Python Reference

Functions

  • MessageBox(message, caption = 'Message')

    Displays a popup dialog box containing the message specified.
    An Ok button is placed on the box.

  • Log(message)

    Writes a string to ika.log, if logging is enabled.

  • ShowDialog(columns -> int, caption -> string, elements -> sequence) -> dict

    Shows a dialog to the user. columns is the number of elements to show
    in one vertical column.

    Elements is a sequence containing two-tuples of strings in the format
    ('name', 'type')
    Currently, the only valid type is 'edit', which creates a text box.

    The return value is a dict containing string results for each element
    passed. The value depends on the type of control created.

    Edit boxes return their contents.

Objects

Map

Represents the current map

  • Map.layercount

    Gets the number of layers on the current map.

  • Map.title

    Gets or sets the map's title.

  • Map.GetMetaData() -> dict

    Creates a dictionary containing the map's metadata, and returns it.

  • Map.tilewidth

    Gets the width of the current tileset

  • Map.GetLayerName(layerIndex) -> string

    Returns the name of the layer with the given index.

  • Map.tileheight

    Gets the height of the current tileset

  • Map.GetTile(x, y, layer) -> int

    Returns the index of the tile at (x, y) on the layer specified.

  • Map.tilecount

    Gets the number of tiles in the current tileset

  • Map.FindLayerByName(name) -> integer

    Returns the index of the first layer with the given name, or None if no such layer
    exists.

  • Map.GetObs(x, y, layerIndex) -> int

    Returns 1 if the tile at (x, y) is obstructed, or 0 if not.

  • Map.SetParallax(layer, xmul, xdiv, ymul, ydiv)

    Sets the specified layer's parallax settings according to the multipliers
    and divisors given. If either of the divisors are zero, a parallax value of 0/1
    will be used for that axis.

  • Map.SetObs(x, y, layerIndex, obs)

    If obs is nonzero, the tile at (x, y) is obstructed, else it is
    unobstructed.

  • Map.GetLayerSize(layerIndex) -> (x,y)

    Returns a tuple containing the size of the layer.

  • Map.GetParallax(layer) -> (int, int, int, int)

    Returns a 4-tuple containing parallax settings for the layer specified.
    Its contents are as follows:
    ( parallax X multiplier,
    parallax X divisor,
    parallax Y multiplier,
    parallax Y divisor )

  • Map.SetTile(x, y, layer, tile)

    Sets the tile at (x, y) of the layer specified.

  • Map.SetMetaData(metaData)

    Copies the dict into the map's metadata. The dict must contain nothing
    but strings for both keys and values.

Editor

Methods and properties pertaining to the ikaMap editor itself.

  • Editor.curlayer

    Gets or sets the active map layer.

  • Editor.GetMouseState() -> (x, y, left, middle, right)

    Returns a tuple containing the status of the mouse.
    x and y are in map pixel coordinates. Left, middle, and right
    are boolean values representing the current state of each mouse
    button.

  • Editor.curtile

    Gets or sets the current tile.

  • Editor.xwin

    Gets the current x position of the map viewport.

  • Editor.TileToMap(x, y, layer=-1) -> (x,y)

    Translates the point (x,y) from tile coordinates on the layer
    specified to world pixel coordinates. If the layer is omitted
    or is an invalid value, the current layer is used.

  • Editor.MapToTile(x, y, layer=-1) -> (x,y)

    Translates the point (x,y) from map pixel coordinates to tile
    coordinates on the layer specified. If the layer is omitted,
    or is an invalid value, the current layer is used.

  • Editor.ywin

    Gets the current y position of the map viewport.
